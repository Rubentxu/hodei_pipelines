# Tech Context: Hodei Pipelines

## Stack Tecnológico

- **Lenguaje:** Kotlin
- **Framework de Backend:** Ktor (para el worker y la posible API de gestión)
- **Scripting:** Kotlin Scripting (`kotlin-script-runtime`, `kotlin-script-util`)
- **Testing:** JUnit 5, MockK, Kotlin Test
- **Contenedores:** Docker (para la ejecución de tareas y el empaquetado de la aplicación)
- **Construcción:** Gradle

## Herramientas de Calidad de Código

- **Linter/Formateador:** ktlint
